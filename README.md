# Hugo Ehrman — Portfolio Site

Personal portfolio for Hugo Ehrman, gameplay programmer at Starbreeze Studios.

**Live site:** https://zleebo.github.io/portfolio-site

## Stack

Plain HTML, CSS, and JavaScript — no framework, no build step.

- `index.html` — structure and content
- `style.css` — dark theme, CSS custom properties, responsive grid
- `script.js` — project card rendering, modal system, scroll animations
- `assets/` — video demos, thumbnails, CV

## Running locally

Open `index.html` in a browser. No server required.
